Google Analytics

What is Google Analytics?

How does Google Analytics work?

Google Analytics includes features:

  • data visualization and monitoring tools
  • data filtering
  • data collection application program interfaces
  • predictive analytics, intelligence
  • segmentation for analysis of subsets
  • custom reports for advertising
  • email-based sharing
  • integration with other products

Important metrics

  • Users
  • Bounce rate
  • Sessions
  • Average session duration
  • Percentage of new sessions
  • Pages per session
  • Goal completions
  • Page views
